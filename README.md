### Week2-2 Git
> **Git 상태확인 명령어**  
git show  
git log  
git shorlog  
git diff  
git status
* **Git 필수 명령**
  * add: commit할 목록에 추가
  > git add ~
  * commit: commit 만들기
  > git commit -m "~"
  * push: 현재까지의 commit을 Github(원격저장소)에 밀어넣기
  > git push origin master
